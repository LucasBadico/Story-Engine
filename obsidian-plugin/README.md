# Story Engine Obsidian Plugin

Connect to Story Engine backend to manage and generate stories directly from Obsidian.

## Features

- List all stories in your workspace
- Create new stories
- View story details
- Clone stories to create new versions

## Setup

1. Install dependencies: `npm install`
2. Build the plugin: `npm run build` or `npm run dev` (watch mode)
3. Link the plugin to your Obsidian vault (see DEVELOPMENT.md)
